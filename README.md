# Developer Portfolio

A personal developer portfolio built using Next.js, TailwindCSS, Framer Motion, and tsparticles.
Includes a terminal-style intro, glass UI cards, smooth animations, and a responsive layout.

## Tech Stack

* Next.js
* React
* TailwindCSS
* Framer Motion
* tsparticles
* shadcn/ui

## Setup

```bash
git clone https://github.com/jhapriyansh/portfolio
cd portfolio
npm install
npm run dev
```

Open:

```
http://localhost:3000
```

## Build

```bash
npm run build
npm start
```
