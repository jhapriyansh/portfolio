# Developer Portfolio
A personal developer portfolio built using Next.js, TailwindCSS, Framer Motion, and tsparticles, featuring a terminal-style intro, glass UI cards, smooth animations, and a responsive layout.

## Features
* Terminal-style intro
* Glass UI cards
* Smooth animations
* Responsive layout
* About, Contact, Projects, Resume, and Skills sections
* Particles background for visual effects
* Customizable UI components

## Tech Stack
* Next.js
* React
* TailwindCSS
* Framer Motion
* tsparticles
* @radix-ui/react-separator
* @radix-ui/react-slot
* @tsparticles/react
* @tsparticles/slim
* class-variance-authority
* clsx
* lucide-react
* tailwind-merge

## Installation
1. Clone the repository: `git clone https://github.com/jhapriyansh/portfolio`
2. Change into the project directory: `cd portfolio`
3. Install dependencies: `pnpm install`
4. Start the development server: `pnpm run dev`

## Usage
Open `http://localhost:3000` in your web browser to view the portfolio.

## Build
To build the portfolio for production, run:
```bash
pnpm run build
pnpm start
```
